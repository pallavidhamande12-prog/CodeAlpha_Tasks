This repository contains backend projects completed as part of the CodeAlpha Backend Internship.

It includes:

Task 1: URL Shortener
A backend application that converts long URLs into short links and redirects users to the original website.
The system stores URLs in a MySQL database and ensures data persistence with duplicate URL handling.

Task 2: Event Registration System
A backend system for managing events, users, and registrations using RESTful APIs.
It supports event creation, user registration, and registration management with proper database relationships.

Each project is implemented using backend technologies such as Flask, Node.js, Express, MySQL, and Sequelize.

Please refer to the respective project folders for detailed documentation, setup instructions, and API details.
